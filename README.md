# 2024~2025 실전문제연구단 (X-twice)

스마트폰 내장 초고속 카메라 기반 비접촉식 기계 결함 진단 시스템 

## 1. 프로젝트 개요 
* **기간**: 2024.06 - 2025.01
* **참여자**: 김태완(팀장), 이태훈, 김서랑, 최성현 

---

## 2. 개발 환경 

* **OS**: Ubuntu 22.04 및 Windows 10
* **Language**: Python 3.8+ 및 kotlin 1.9.0
* **Hardware**: Intel Core i7-13700 CPU @ 5.2 GHz

---

## 3. 폴더 구조 
```text
├── src/            # 소스 코드
├── data/           # 데이터셋 
├── docs/           # 관련 문서
├── weights/        # 학습된 모델 가중치 
└── README.md
```

---

## 4. 브랜치 구조

* **main**: 메인 브랜치
* **app_integration**: 앱 개발 브랜치
* **sr**: 모델 훈련 관련 브랜치
